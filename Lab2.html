Execution of different functionalities in Seeed Xiao nRF52840 Sense

1. Built-in IMU and Microphone:

Here we are executing the high Level Example from Seeed Arduino "LSM6DS3" Library and Tetsing the different sensors Thermometer, Accelerometer, Gyrosope. Here we are observing the output in serial Monitor

Next we are testing the microphone using PDMSerialPlotter

2. OLED Display:

Here we are executing a sample code to print the text message in display of Seeed Xiao using "u8g2" Library

3. MicroSD Card Datalogger:

Here we are writing and reading data from the SD Card and observing the output in the serial monitor

Positive scenario:

As SD card is inserted in port-2, we will write and read data from port 2. So, we should be able to see the random string data

Negative scenario:

Here we will try to get the data from the port other than 2. So, we should see some failed cases

4. RTC (Real-time clock):

Here we are generating the real time clock using "PCF8563" Library and observing the output through serial monitor

5. Buzzer:

Here we are validating the buzzer by playing the Happy Birthday Music

6. Bluetooth Low Energy:

Here we are using "ArduinoBLE" Library for validating the Bluetooth functionality by sending the data input form mobile app (nRF Connect app) and controling the LED in the Micro Controller
