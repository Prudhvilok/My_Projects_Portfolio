Voltage Divider and Battery Monitorin in Seeed Xiao nRF52840 Sense

1.Circuit Diagram fo Voltage divider

In our case We have taken 3 resistors, so R1 will be 2 resistors in series and R2 will be the separate resistor.

R1 = 680+2200 = 2800 ohms

R2 = 10000 ohms

Vout = Vin*(R2/(R1+R2))

We have Vout = 3.3v and Vin = 4.2v

After calculation we will get the R1/R2 ratio as 0.28. So, we have choosen R1 and R2 bases on this ratio.

2.Battery Monitoring

1. System Workflow:
1. System Initialization:
On startup, the microcontroller initializes the sensors, RTC, SD card module, and OLED display.
The OLED shows the current battery voltage and system status (ON/OFF).

2. Bluetooth Control:
The user connects to the system via Bluetooth and sends commands to turn the system
ON or OFF.
When ON, the system begins recording data; when OFF, it halts data logging but keeps
Bluetooth active.

3. Battery Monitoring:
Continuously reads the battery voltage through the ADC pin.
If the voltage drops below the threshold (e.g., 3.2V), the system:
and Stops logging data.
Displays a warning message on the OLED (e.g., "Low Battery: Logging
Stopped").

4. Data Logging:
While ON, the system:
Reads sensor data (microphone, temperature, IMU).
Timestamps the data using the RTC.
Logs data to the SD card in a structured format (e.g., CSV or JSON).
When OFF, logging is paused, and the OLED updates with the system status.

5. Real-Time Display:
The OLED continuously updates to show:
Current battery voltage.
System status (ON/OFF)
